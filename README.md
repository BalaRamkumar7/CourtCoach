## Overview
CourtCoach is a personal project looking into how AI and computer vision can replicate the experience of having a real basketball coach without needing one. The app uses your phone's camera to watch you practice, analyzes your body mechanics in real time, delivers personalized spoken coaching tips as you drill, and generates a full session summary when you're done. This app is an exploration of mobile development, AI integration, and pose-tracking systems using real-world sports biomechanics data.

## Objectives
* Build a real-time coaching experience that tells players not just what happened, but how to fix it
* Integrate pose tracking to extract meaningful basketball mechanics from raw camera input
* Apply AI-generated natural language to translate biomechanics data into actionable, personalized coaching tips
* Design a session memory system that accumulates feedback across a drill and summarizes performance at the end
* Develop a scalable drill and skill structure covering shooting, passing, and dribbling
* Implement a clean mobile interface that feels intuitive for players of all ages

## Methodology
* Capture live camera feed from the player's device during a drill
* Extract body keypoints (shoulder, elbow, wrist, knee, ankle) using MediaPipe pose tracking
* Calculate basketball-specific metrics from keypoints — elbow angle, knee bend, release height, balance score, follow-through
* Evaluate metrics against expert-defined ideal ranges for each drill type
* Send metric deviations to Claude API, which translates them into plain-language coaching tips
* Deliver tips via on-screen overlay and spoken audio using expo-speech
* Accumulate tips in SessionContext throughout the drill
* On session end, send full tip history to Claude for a structured summary (rating, top strength, main focus, encouragement)
* Speak and display the summary on the Feedback screen
